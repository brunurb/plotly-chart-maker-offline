ChartMaker - CSV to Chart Converter

To run this application:

1. Install Docker Desktop from https://www.docker.com/get-started
2. Start Docker Desktop
3. Double-click on:
   - run.bat (Windows)
   - run.sh (macOS/Linux)
4. Wait for the application to start (it may take a few minutes on first run)
5. Your default web browser will open to http://localhost:8501
6. Exported charts will be saved in the 'output_charts' folder

Note: Docker must be installed and running.
